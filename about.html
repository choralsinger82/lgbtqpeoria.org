About this project

LGBTQ Peoria is a local resource hub for Peoria and Central Illinois, starting with Rainbow Pages — a community-reviewed directory of affirming spaces.

Inclusivity

We use “LGBTQ” as a widely recognized shorthand. This site explicitly includes lesbian, gay, bisexual, transgender, queer, intersex, asexual, aromantic, nonbinary, questioning, and gender-diverse people.

If language on the site can be improved, we want to hear that. This is meant to evolve with the community.

Contact

To request changes, report concerns, or ask questions, email: lgbtqpeoria@gmail.com

Disclaimer

Listings are informational and not endorsements. If you notice inaccurate information, changes in behavior, or safety issues, please contact us so we can review and update the directory.
